# Object Synth

Object Synth is a camera-driven audio instrument: movement in on-screen zones triggers sounds.

This repository includes a non-technical guide for performers and workshop users:

- [`User Guide`](docs/user-guide.md) - setup, core workflow, tuning tips, and troubleshooting.

## Local Development

- Install dependencies: `pnpm install`
- Start dev server: `pnpm dev`
- Build production bundle: `pnpm build`
